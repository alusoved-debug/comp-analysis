# אפליקציית פתקים חכמים

אפליקציית פתקים פשוטה ב-HTML/CSS/JavaScript שמאפשרת:

- יצירת פתקים עם טקסט
- שימור תמונות וקבצים מצורפים
- ציור על קנבס ושמירת הציור כפתק
- חיפוש פתקים ושמירתם בדפדפן באמצעות `localStorage`

## שימוש

1. פתח את `index.html` בדפדפן.
2. כתוב כותרת ותוכן לפתקים.
3. לחץ על `שמור פתק` כדי לשמור.
4. העלה קבצים או תמונות באמצעות לחצן הבחירה.
5. צייר על הקנבס ולחץ על `שמור ציור כפתק` כדי לשמור את הציור כפתק חדש.
6. חפש פתקים בשדה החיפוש או מחק פתקים לפי הצורך.

## פיתוח

אין צורך בהתקנות נוספות. אם תרצה להרחיב את האפליקציה, הוסף:

- סיווג פתקים
- עריכה של פתקים קיימים
- סנכרון ענן
- תצוגה מודאלית של תמונות
